=========================
 Installation
=========================

This tutorial walks you through how to set up glitter-sdk-js_ for local development.

The glitter Node Driver depends on:

1. Node v14.0.0+
2. A recent Node version of npm

Installing with npm
---------------------

We recommend using npm to install glitter_sdk on all platforms:

.. code-block:: bash

    $ npm install glitter-sdk-js

To get a specific version of glitter:

.. code-block:: bash

    $ npm install glitter-sdk-js@0.1.4

To upgrade using npm:

.. code-block:: bash

    $ npm update glitter-sdk-js
